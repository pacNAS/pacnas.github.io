Why pacNAS?

pacNAS is a NAS distribution offering the whole stack of functionality for a simple NAS system out of the box. The pacNAS community wants to offer a NAS distribution similar to other NAS distributions, but with some slight differences not offered from any other NAS distribution (at least right now).

pacNAS will provide btrfs as the primary file system for the data storeage. You can expect to have an easy Installer at hand and also a webbased Administration tool (Ajenti).

Freedom Of Choice

pacNAS is based on the ArchLinux distribution and therefor provides the famous Pacman package manager. Due to this, you will have access to a whole lot of packages. If you are interested in more packages, you can optionally install one of the AUR Helpers (e.g. packer) and access the ArchUserRepository.

Status

pacNAS is an unofficial community project. It is still in development. There is currently no stable release suitable for production usage yet. This project is not related to the official ArchLinux project other then we are using their packages as a base to build this distribution.
